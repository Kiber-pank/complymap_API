# Complymap_API

Сервер для работы с данными деклараций и сертификатов complymap_API. Предоставляет REST API с фильтрацией, пагинацией и подтягиванием справочников через JOIN.

## Стек технологий

- Node.js 20+
- TypeScript
- Fastify
- PostgreSQL 17+
- Zod (валидация запросов)
- Pg (пул соединений)

## Требования

- Node.js версии 20 или выше
- PostgreSQL 17 или выше
- Настроенные переменные окружения (см. раздел ниже)

## Установка и запуск

1. Склонируйте репозиторий:
```bash
git clone https://github.com/Kiber-pank/complymap_API.git
cd fsa-api
```
2. Установите зависимости:
```npm install```

3. Cоздайте файл окружения:
```cp .env.example .env```
Отредактируйте .env, указав реальные доступы к базе данных.

4. Запустите сервер в режиме разработки:
```npm run dev```
Сервер будет доступен по адресу http://localhost:3000.

## Переменные окружения

| Переменная    | Описание                 | Пример                        |
| ------------- |:------------------------:| -----------------------------:|
| PG_HOST       | Адрес сервера PostgreSQL |  localhost или db.example.com |
| PG_PORT       | Порт PostgreSQL          |                          5432 |


## Структура проекта

src/
├── config/           # Валидация переменных окружения
├── db/               # Настройка пула соединений PostgreSQL
├── modules/
│   └── declarations/ # Маршруты, сервисы и схемы для деклараций
├── plugins/          # Глобальные плагины (обработчик ошибок, rate-limit)
├── routes/           # Системные маршруты (healthcheck)
├── app.ts            # Сборка приложения Fastify
└── server.ts         # Точка входа и запуск HTTP-сервера
